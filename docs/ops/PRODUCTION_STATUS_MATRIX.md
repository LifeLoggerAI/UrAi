# URAI Production Status Matrix

Scoring:

0 = missing  
1 = repo exists only  
2 = source exists, no deploy proof  
3 = source plus tests  
4 = deploy config  
5 = deployed surface responds  
6 = smoke tests pass  
7 = rollback plus monitoring  
8 = privacy/security gates pass  
9 = owner/legal signoff  
10 = production certified
