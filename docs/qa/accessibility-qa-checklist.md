# URAI Accessibility QA Checklist

- [ ] Keyboard navigation works across Genesis and overlays.
- [ ] Focus trap works in modal-style views.
- [ ] Escape closes the topmost layer where expected.
- [ ] Buttons have labels.
- [ ] Toggles expose state.
- [ ] Sliders have accessible names.
- [ ] Reduced motion is respected.
- [ ] Reduced sensory mode is available.
- [ ] Captions work when voice is off.
- [ ] No sound-only information.
- [ ] No color-only status.
- [ ] Decorative images use empty alt text.
- [ ] Major modals have aria labels or labelled headings.
- [ ] Destructive actions require confirmation.
- [ ] Mobile close buttons are reachable.
