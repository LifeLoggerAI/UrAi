# URAI Device QA Checklist

## Mobile

- [ ] iPhone SE / small screen.
- [ ] iPhone standard.
- [ ] iPhone Pro Max.
- [ ] Android compact.
- [ ] Android tall.

## Tablet

- [ ] iPad portrait.
- [ ] iPad landscape.
- [ ] Android tablet.

## Desktop

- [ ] 1366 x 768.
- [ ] 1440 x 900.
- [ ] 1920 x 1080.
- [ ] Ultrawide sanity check.

## Browsers

- [ ] Safari iOS.
- [ ] Chrome Android.
- [ ] Chrome desktop.
- [ ] Safari desktop.
- [ ] Firefox desktop.
- [ ] Edge desktop.

## Checks

- [ ] No clipped orb.
- [ ] Ground reachable.
- [ ] Passport reachable.
- [ ] Settings reachable.
- [ ] Life Map usable.
- [ ] Companion input usable.
- [ ] Keyboard does not cover input badly.
- [ ] Modals close correctly.
- [ ] Focus returns correctly.
- [ ] No horizontal overflow.
- [ ] Reduced sensory mode works.
