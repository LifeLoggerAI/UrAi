This evidence gate intentionally blocks public claims that a data marketplace or marketplace payouts are live.

Allowed public language: roadmap, gated, private beta, waitlist, aggregate-only foundation, coming soon, and not live.

To unlock a public production claim, attach separate marketplace opt-in evidence, aggregate privacy review, minimum cohort proof, legal approval, live smoke evidence, revoke/exclusion proof, and rollback proof.
