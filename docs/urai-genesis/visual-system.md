# URAI Genesis Visual System

## Core visual identity

URAI Genesis must feel magical, cinematic, emotional, polished, minimal, soft, alive, symbolic, premium, and non-clinical.

It must not feel like a SaaS dashboard, medical chart, generic wellness app, chatbot wrapper, prototype, or surveillance product.

## World palettes

| World | Visual direction |
|---|---|
| Home | Deep blue sky, green/earth ground, soft silver-blue orb, warm bloom highlights |
| Galaxy | Deep-space navy, luminous white stars, emotional color accents |
| Mirror | Glass, cool silver, soft reflection gradients |
| Shadow | Safe darkness, violet-blue, low contrast, no harsh red |
| Legacy | Amber, gold, parchment glow, cinematic warmth |
| Passport | Clean white-blue, secure shimmer, lock/sigil language |
| Council | Chamber glow, quiet radial light, orb resonance |

## Mood / weather visuals

| State | Visual |
|---|---|
| Clear | Soft blue-gold sky, slow particles |
| Quiet | Muted blue, low motion, dim stars |
| Heavy | Deeper sky, slow fog, low horizon glow |
| Bright | Gold/white particles, lifted sky |
| Restless | Faster particle drift, shifting wind |
| Focused | Clean beam, stable orb pulse |
| Overstimulated | Denser particles, subtle flicker |
| Recovering | Green/gold bloom, warmer ground |
| Unknown | Neutral dusk, low motion |

## Motion rules

Motion is slow, breathing, atmospheric, and purposeful.

Every motion must serve one of these purposes:

- reveal
- calm
- connect
- explain
- transition
- confirm control

Avoid twitchy, gamified, cheap, random, or excessive animation.

## UI rules

Use:

- glass layers
- soft glows
- particles
- rounded panels
- minimal copy
- cinematic transitions
- glowing symbolic objects

Avoid:

- generic cards
- dense dashboards outside Passport/Settings
- raw scores
- medical-looking charts
- hard red warning states unless safety-critical
- debug labels
- placeholder UI

## Accessibility visual requirements

- Reduced motion must replace major transitions with fades.
- High contrast mode must preserve hierarchy and readability.
- Sound can never be required to understand a state.
- Color alone must never be the only state indicator.
