# URAI Sensitive Language Audit

Use this before launch and after major copy changes.

## Avoid or review carefully

- diagnose
- treat
- cure
- proof
- always
- never
- we know
- URAI knows
- surveillance
- lie detector
- betrayal certainty
- medical advice
- emergency support
- unlock your full profile
- recommended for best experience on sensitive permissions

## Replace with

- may suggest
- might reflect
- from what is visible
- if you choose
- Passport controls this
- summary only
- not a verdict
- you can close this

## Required copy anchors

- This is a reflection, not a verdict.
- Nothing is carried forward unless you choose.
- Review what leaves URAI.
- URAI should whisper only when it matters.
- Sound changes what URAI plays.
- Audio capture controls what URAI may listen to or transcribe.
- Sync copies only approved URAI state to your account.
- This changes data URAI controls. It does not delete data from third-party services.

## Audit checks

- [ ] No diagnosis or treatment claim appears in user-facing copy.
- [ ] No deception certainty appears in relationship or Shadow copy.
- [ ] No sensitive permission uses pressure language.
- [ ] No closed layer is described as if URAI can see it.
- [ ] No export copy implies automatic sharing.
- [ ] No notification copy reveals Shadow, health, location, Gmail, calendar, transcript, or relationship details.
