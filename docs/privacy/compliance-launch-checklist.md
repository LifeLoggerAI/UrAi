# URAI Compliance Launch Checklist

This checklist supports product and engineering launch review. It does not replace professional legal review.

## Passport

- [ ] Sensitive layers default off.
- [ ] Safe defaults verified.
- [ ] Explicit opt-in verified for Shadow, Legacy, Export, Audio Capture, Transcripts, Location, Gmail, Calendar, Health, Relationships, Companion memory, and sensitive cloud sync.
- [ ] Consent copy is not manipulative.

## AI

- [ ] Closed layers are blocked.
- [ ] Raw sensitive context is not sent by default.
- [ ] No diagnosis claims.
- [ ] No deception certainty.
- [ ] No raw source IDs.
- [ ] No raw Gmail, calendar, transcript, location, or health data.

## Shadow

- [ ] Off by default.
- [ ] Consent gate present.
- [ ] Ground exit available.
- [ ] Soft mode default.
- [ ] Copy says: This is a reflection, not a verdict.

## Legacy

- [ ] Off by default.
- [ ] User approval required.
- [ ] No automatic Companion memory.
- [ ] Copy says: Nothing is carried forward unless you choose.

## Exports

- [ ] Off by default.
- [ ] Review required.
- [ ] Hidden or sealed items blocked.
- [ ] Shadow export requires extra confirmation.
- [ ] No private IDs in filenames.

## Notifications

- [ ] Off by default or explicitly limited.
- [ ] No sensitive text.
- [ ] No Shadow content.
- [ ] Push not requested on first load.
- [ ] Quiet hours enabled by default.

## Audio

- [ ] Playback separated from capture.
- [ ] Microphone not requested unexpectedly.
- [ ] Transcript permission is explicit.
- [ ] Sound playback copy does not imply recording.

## Cloud

- [ ] Sync is opt-in.
- [ ] Sign-in required for cloud sync.
- [ ] Sensitive sync off by default.
- [ ] Most restrictive conflict setting wins.

## Account

- [ ] Deletion flow is clear.
- [ ] Local/cloud distinction is clear.
- [ ] Third-party deletion is not overclaimed.

## Accessibility

- [ ] Privacy controls are keyboard accessible.
- [ ] Consent state is not color-only.
- [ ] Cancel path is visible.

## Legal review

- [ ] Privacy policy reviewed.
- [ ] Terms/product boundaries reviewed.
- [ ] Data retention reviewed.
- [ ] Age/minor policy reviewed.
- [ ] Third-party integrations reviewed.
