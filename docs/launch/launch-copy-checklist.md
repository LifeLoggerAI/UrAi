# URAI Launch Copy Checklist

Check all launch, demo, waitlist, and social-preview copy for:

- [ ] No diagnosis claims.
- [ ] No therapy replacement claims.
- [ ] No “always listening” language.
- [ ] No “lie detector” language.
- [ ] No “knows you better than you” claims.
- [ ] No overpromising AI certainty.
- [ ] Privacy-first copy is visible.
- [ ] Passport explanation is present.
- [ ] Demo sample-data disclosure is present.
- [ ] Waitlist consent is clear.
- [ ] Founder story is ready and does not imply private memories are shown.
- [ ] Shadow is described as protected, not scary.
- [ ] Legacy is described as chosen memory, not automatic memory.
- [ ] Export copy says review is required.
