# Public Claims Audit

Use this checklist before publishing launch, demo, waitlist, fundraising, investor, social, or press copy.

## Avoid

- “diagnoses”
- “treats”
- “detects lies”
- “knows how you feel”
- “always listening”
- “predicts mental health”
- “guarantees recovery”
- “replaces therapy”
- “sells your data”
- “fully anonymous” unless technically and legally true

## Use

- “may help reflect”
- “permissioned”
- “sample data”
- “private by default”
- “user controlled”
- “symbolic”
- “not a medical device”

## Required checks

- [ ] Demo copy says sample data.
- [ ] Privacy copy is visible before sensitive claims.
- [ ] Passport is described as the control layer.
- [ ] No professional-care claims.
- [ ] No certainty claims about another person.
- [ ] No hidden access to email, location, transcripts, health, or relationships.
- [ ] No promise that exports happen automatically.
- [ ] No fundraising language promises outcomes or equity.
