# URAI Privacy Promise

## Core promise

URAI is private by default. Passport controls what opens.

## What that means

- Sensitive layers start closed.
- AI uses only permissioned summaries.
- Exports require review.
- Shadow and Legacy are opt-in.
- Notifications are off by default.
- Audio playback is not audio recording.
- Local-only mode is supported.
- Users can close layers anytime.

## Short version

Nothing meaningful opens unless you choose to open it.
