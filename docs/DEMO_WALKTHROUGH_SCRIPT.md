# URAI Demo Walkthrough Script

## Goal

A first-time viewer should understand URAI in under three minutes.

## Scene 1 — The problem

Modern life is scattered across messages, photos, notes, calendars, files, and memories.

## Scene 2 — The idea

URAI turns personal context into a spatial AI memory world.

## Scene 3 — The world

Show the spatial interface:

- Home
- Ground
- Life Map
- Focus
- Replay
- Mirror
- Passport
- Status

## Scene 4 — The intelligence layer

Explain that AI can organize context, surface relationships, and help people navigate information through a privacy-first interface.

## Scene 5 — The proof

Show:

- repositories;
- architecture docs;
- release evidence;
- privacy boundaries;
- public receipts.

## Closing line

URAI is building a new interface between people, memory, and AI: a spatial layer for personal context.

## Demo discipline

Show what exists. Label what is future. Never imply a capability without evidence.
