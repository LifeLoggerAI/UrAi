# Consent and Passport Map

This is not legal advice. This document maps consent points to the Passport system.

The URAI Passport system is designed to manage user consent for sensitive features.

* **Initial Onboarding**: High-level consent for core features.
* **Sensitive Feature Gates**: Explicit opt-in required for features like:
    * AI Companion chat analysis
    * Microphone access for transcription
    * Integration with third-party services (e.g., Google Calendar)
* **Revocation**: Users can revoke consent at any time through the Passport settings.
