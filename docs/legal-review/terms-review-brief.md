# Terms Review Brief

This is not legal advice.

URAI Genesis should be described as a private symbolic life interface for reflection and meaning-making through a visual demo experience.

## Product Boundaries

URAI should be described as reflection, not diagnosis. It should not be presented as medical care, emergency support, clinical validation, lie detection, or guaranteed outcome software.

## Public Demo

The public demo uses sample data only. Demo content is illustrative and not based on a real person's private life.

## Review Need

Counsel should review final terms, disclaimers, user responsibilities, acceptable use, feedback ownership, and platform language.
