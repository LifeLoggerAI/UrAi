# Data Map

This is not legal advice. This map is for discussing data flows with counsel.

* **Waitlist**: Email collected, stored in Firebase.
* **Feedback**: User-submitted content, stored in Firebase.
* **Core App**: User profile, settings, journal entries, etc., stored in Firebase.
* **Analytics**: Usage data, collected via Firebase Analytics.
* **Exports**: User-generated data exports, temporarily stored for download.
