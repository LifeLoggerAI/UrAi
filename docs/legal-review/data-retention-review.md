# Data Retention Review

This is not legal advice.

## Public Demo

Demo data is part of the application and does not contain user information. No private data retention to review for the public demo.

## Waitlist

Waitlist data should be retained only as long as needed to manage the waitlist. Counsel should review retention and deletion policies.

## Feedback

Feedback may be retained for product improvement. Review whether feedback should be anonymized and for how long it should be kept.

## Future Private Data

Data retention policies for future private data will require careful review. This includes Companion memory, integration data, and user-generated content.

## Deletion

Users should have a clear path to request deletion of their waitlist information or future private data. The deletion process must be reviewed by counsel.
