# URAI Genesis Mock Matching QA Checklist

Use this checklist after final Genesis image assets are dropped into `public/assets/genesis`.

## Mobile

- [ ] iPhone small viewport checked.
- [ ] iPhone Pro Max viewport checked.
- [ ] Android tall viewport checked.
- [ ] Android compact viewport checked.
- [ ] Body remains centered and not stretched.
- [ ] Orb is reachable and not cropped.
- [ ] Ground does not swallow the body.
- [ ] Passport remains visible and reachable.
- [ ] Settings glyph remains visible and reachable.

## Tablet

- [ ] iPad portrait checked.
- [ ] iPad landscape checked.
- [ ] Sky portal is not cut off.
- [ ] Body crop still matches the approved mock direction.
- [ ] Ground height feels cinematic, not like a footer.
- [ ] Portal objects feel environmental, not like dashboard buttons.

## Desktop

- [ ] 1366x768 checked.
- [ ] 1440x900 checked.
- [ ] 1920x1080 checked.
- [ ] Ultrawide checked.
- [ ] Sky, body, orb, Ground, and vignette keep the approved composition.
- [ ] No layer appears pixelated at desktop size.

## Visual Quality

- [ ] Sky matches approved mock direction.
- [ ] Cloud depth feels layered.
- [ ] Mood overlays are subtle and emotional, not noisy.
- [ ] Body layers are aligned.
- [ ] Aura layers are aligned.
- [ ] Orb uses final image assets, not a CSS circle.
- [ ] Ground uses final image assets.
- [ ] Portals use final image assets.
- [ ] Vignette feels cinematic and does not hide controls.
- [ ] No broken images are visible.
- [ ] No debug labels appear.
- [ ] No placeholder, mock, test, coming-soon, or missing-asset text appears.
- [ ] Scene does not feel like a dashboard.
- [ ] Text remains readable where visible.

## Reduced Sensory Mode

- [ ] Reduced motion disables heavy pulses.
- [ ] Particle shimmer is subdued.
- [ ] Starfield appears static.
- [ ] Opacity changes are softened.
- [ ] No essential interaction is removed.

## Interaction

- [ ] Sky opens Life Map.
- [ ] Ground opens Ground.
- [ ] Orb opens Companion.
- [ ] Passport opens Passport.
- [ ] Settings glyph opens Settings.
- [ ] Keyboard focus is visible only on focus.
- [ ] Interaction zones are invisible.
